# [] WITH US

## 서비스 소개

---
다같이 도형을 만들고 AI가 맞추는 아이스브레이킹 게임, With Us!  

> With Us에서 몸을 움직이며, 사람들과의 어색함을 풀어보아요!  

프로젝트 기간: 2023.07.04 ~ 2023.08.18
